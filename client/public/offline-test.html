Симуляция оффлайн-режима

  1. Откройте инструменты разработчика браузера (F12)
  2. Перейдите на вкладку Network (Сеть)
  3. Установите флажок "Offline" (Оффлайн)
  4. Перезагрузите страницу для проверки работы в оффлайн-режиме
Устранение проблем

Service Worker не регистрируется

  • Убедитесь, что ваш сайт обслуживается через HTTPS (кроме localhost)
  • Проверьте, что файл service-worker.js доступен и не содержит ошибок
  • Очистите кэш браузера и данные сайта

Ресурсы не кэшируются

  • Проверьте список CORE_ASSETS в service-worker.js
  • Убедитесь, что все пути к ресурсам корректны
  • Проверьте консоль на наличие ошибок CORS

Оффлайн-страница не отображается

  • Убедитесь, что оффлайн-страница кэширована
  • Проверьте, что путь к оффлайн-странице в service-worker.js совпадает с фактическим
  • Проверьте обработчик fetch-событий в service-worker.js
Консольное API 
// Проверка поддержки оффлайн-режима
window.appDebug.checkOfflineSupport()

// Очистка кэша Service Worker
window.appDebug.clearServiceWorkerCache()

// Обновление кэша
window.appDebug.refreshServiceWorkerCache()

// Проверка, находится ли приложение в оффлайн-режиме
window.appDebug.isOffline()

// Полный сброс кэша приложения
window.appDebug.resetAppCache()
